# Omarchy Dotfiles

My Omarchy Linux configuration managed with GNU Stow.

## Restore on fresh install
```bash
cd ~
git clone git@github.com:justincarlson/omarchy-dotfiles.git dotfiles
cd dotfiles
stow omarchy-config
```

## Managed configs

- Hyprland (window manager)
- Waybar (top bar)
- Walker (launcher)
- Ghostty (terminal)
- uwsm (session manager)
- Starship (shell prompt)
- .bashrc and .XCompose
